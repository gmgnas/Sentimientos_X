Documento de Arquitectura de Solución

Sistema: Sentimientos_X | Versión 1.0

1. Definición de la Arquitectura

La solución se basa en una arquitectura de Pipeline de Datos Desacoplado. Los componentes operan de forma independiente, asegurando que el sistema sea adaptable a cambios en las fuentes de datos o en las reglas de negocio sin comprometer la integridad global.

2. Recursos Técnicos

  • Hardware: Estación de trabajo o VPS con soporte Python 3.10+.
  • OS: Linux (Pop!_OS/Ubuntu) o Windows.
  • Almacenamiento: SQLite para portabilidad absoluta.

3. Seguridad y Privacidad

Implementación de estándares de seguridad industrial:

  • Tokens: Gestión mediante archivos .env (fuera del repositorio).
  • Cumplimiento: Alineación con la Ley 25.326 de Protección de Datos Personales.
