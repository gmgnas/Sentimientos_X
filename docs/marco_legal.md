# Marco Legal y Normativo (Argentina)

Dado que trabajamos con datos de usuarios de X (anteriormente Twitter), en Argentina aplican las siguientes normativas:

## Normativas Principales

- **Ley de Protección de Datos Personales N° 25.326**: Norma principal que exige consentimiento del titular (Art. 5) y finalidades determinadas, explícitas y legítimas para el tratamiento de datos, incluso si los tweets son públicos.

- **Declaraciones sobre Data Scraping (2023-2024)**: Argentina considera el scraping no autorizado como violación grave de privacidad.

- **Requisitos de legitimidad**: El carácter público del post es insuficiente; se requiere base legal válida (interés legítimo, consentimiento o interés público).

- **Recomendación de APIs oficiales**: Las autoridades recomiendan usar APIs de X en lugar de scraping directo para mayor control y detección.

## Implicancias Prácticas

- Usar API Oficial de X (v2) en lugar de scraping directo
- Informar al cliente que el análisis debe ser agregado y anonimizado
- No almacenar identificadores reales de usuarios sin consentimiento explícito
